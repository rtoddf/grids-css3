Grid - Line Based Spanning Tracks

To create Grid Areas that are larger than a single grid track we specify an end line that is more than one track away.

Here I am using the grid-column and grid-row shorthand and have omitted the end value for any items that span one row or column track.

A
B
C
D
.grid {
	display: -ms-grid;
	display: grid;
	grid-gap: 10px;
}

.a {
	-ms-grid-column: 1 / 3;
	-ms-grid-row: 1;
	grid-column: 1 / 3;
	grid-row: 1;
}
.b {
	-ms-grid-column: 3 ;
	-ms-grid-row: 1 / 3;
	grid-column: 3 ;
	grid-row: 1 / 3;
}
.c {
	-ms-grid-column: 1;
	-ms-grid-row: 2;
	grid-column: 1;
	grid-row: 2;
}
.d {
	-ms-grid-column: 2;
	-ms-grid-row: 2;
	grid-column: 2;
	grid-row: 2;
}
