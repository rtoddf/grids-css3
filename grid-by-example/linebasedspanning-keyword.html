Grid - Line Based Spanning Tracks with Span Keyword

We can also span using the span keyword. This example creates the same layout as the one in example 5. I am using the span keyword rather than targeting the Grid Line by number. I am also using the defaults for row and column end, which is to span 1.

A
B
C
D
.grid {
	display: -ms-grid;
	display: grid;
	grid-gap: 10px;
}

.a {
	-ms-grid-column: 1 / span 2;
	-ms-grid-row: 1;
	grid-column: 1 / span 2;
	grid-row: 1;
}
.b {
	-ms-grid-column: 3;
	-ms-grid-row: 1 / span 2;
	grid-column: 3;
	grid-row: 1 / span 2;
}
.c {
	-ms-grid-column: 1;
	-ms-grid-row: 2;
	grid-column: 1;
	grid-row: 2;
}
.d {
	-ms-grid-column: 2;
	-ms-grid-row: 2;
	grid-column: 2;
	grid-row: 2;
}
