Grid - Line Based

Using the Grid I defined in example 1, I am positioning the elements in my markup (six divs with a class of grid-item and classes from a to f) using line-based placement properties. I've also added a background color to help visualize the boxes. This example is more verbose than it needs to be as a demonstration of the properties. In reality if an item will only span one grid track you may omit the -end value.
